# Contributing

Thank you for your interest in contributing to ui-audit.

## Code of Conduct

Please read and follow our [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md) in all interactions.

## Development Setup

```bash
npm install
npm run build
npm run test
npm run lint
```

## Pull Request Checklist

- Keep changes focused and well documented
- Add or update tests when behavior changes
- Ensure linting and build checks pass
- Follow the existing code style

## Reporting Issues

Please use the issue templates under [.github/ISSUE_TEMPLATE](.github/ISSUE_TEMPLATE) for bug reports and feature requests.
