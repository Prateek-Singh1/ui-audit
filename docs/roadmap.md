# Roadmap

## Short term

- Stabilize the CLI interface and developer experience
- Add static scanning for common frontend patterns
- Add accessibility-focused checks

## Medium term

- Add performance heuristics and reporting improvements
- Support additional frameworks and templating patterns
- Improve output formatting for CI and editor integrations

## Long term

- Introduce optional remote integrations behind explicit configuration
- Expand reporting with richer summaries and suggestions
- Grow the contributor ecosystem around the project
